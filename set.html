Set GitSearch as Default Search Engine

Instructions

Follow the instructions below to set GitSearch as your default search engine in different browsers:

Google Chrome

  1. Open Chrome and go to Settings.
  2. Scroll down to the "Search engine" section.
  3. Select "Manage search engines."
  4. Add a new search engine with the following details:
    • Name: GitSearch
    • Keyword: gitsearch
    • URL: https://www.gitsearch.com/search?q=%s
  5. Hover over GitSearch and click "Make default."

Mozilla Firefox

  1. Open Firefox and go to Preferences.
  2. Scroll down to the "Search" section.
  3. Choose "Add Search Engine" and enter the following details:
    • Name: GitSearch
    • Keyword: gitsearch
    • URL: https://www.gitsearch.com/search?q=%s
  4. Select GitSearch from the list and click "Set as Default."

Microsoft Edge

  1. Open Edge and go to Settings.
  2. Scroll down to the "View advanced settings" section.
  3. Under "Search in the address bar with," choose "Add new."
  4. Enter the following details:
    • Name: GitSearch
    • Keyword: gitsearch
    • URL: https://www.gitsearch.com/search?q=%s
  5. Select GitSearch from the list and click "Add as default."

Safari

  1. Open Safari and go to Preferences.
  2. Click on "Search" and choose "Manage Search Engines."
  3. Click "+" to add a new search engine with the following details:
    • Name: GitSearch
    • Keyword: gitsearch
    • URL: https://www.gitsearch.com/search?q=%s
  4. Select GitSearch from the list and click "Set as Default."

GitSearch is now set as your default search engine. Enjoy searching!
